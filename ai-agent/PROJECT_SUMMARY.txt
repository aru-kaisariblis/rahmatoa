📝 COMMANDS OVERVIEW
════════════════════════════════════════════════════════════════════════════════

1. /add ........................ Tambah task baru
   Format: /add Judul | Mata Kuliah | Deadline | Deskripsi
   Example: /add Buat Laporan | Algoritma | 25/03/2026 23:59 | Praktikum

2. /list ....................... Tampilkan tasks
   Options: /list [subject] atau /list [status]
   Example: /list pending, /list Algoritma

3. /done ........................ Tandai task selesai
   Format: /done [task_id]
   Example: /done 1

4. /delete ...................... Hapus task
   Format: /delete [task_id]

5. /subject .................... Lihat mata kuliah

6. /status ..................... Lihat statistik tasks

7. /overdue .................... Lihat tasks yang overdue

8. /help ........................ Tampilkan bantuan

📊 DATA MODELS
════════════════════════════════════════════════════════════════════════════════

Task:
  - id: int
  - title: string (required)
  - description: string
  - subject: string (mata kuliah, required)
  - deadline: timestamp (required)
  - status: enum (pending, in_progress, completed, overdue)
  - priority: enum (low, medium, high)
  - chat_id: string (WhatsApp group ID)
  - user_id: string (User WhatsApp ID)
  - created_at: timestamp
  - updated_at: timestamp

Reminder:
  - id: int
  - task_id: int (foreign key)
  - chat_id: string
  - reminder_time: timestamp
  - message: string
  - is_sent: boolean
  - created_at: timestamp
